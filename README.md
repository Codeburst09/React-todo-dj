# React-todo-dj

## Setup
Install packages for Python:
```
$ pip install -r requirements.txt
```

Setup react.js:
```
cd frontend/
npm install
```

## Usage

Run Django backend:
```
python manage.py runserver
```
OR
```
python3 manage.py runserver
```

Run React.js frontend:
```
cd frontend/
npm start
```

Django backend default port: `8000`\
React.js default port: `3000`
